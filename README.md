
## Starting App

To get started:

* Install and start the API server
    - `cd api-server`
    - `yarn install`
    - `node server`

* In another terminal window, go to front-end:
    - `cd frontend`
    - `yarn install`
    - `yarn start`

## API Server

Information about the API server and how to use it can be found in its [README file](api-server/README.md).
